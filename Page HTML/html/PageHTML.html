C'est quoi HTML en informatique?

L'HyperText Markup Language, HTML, désigne un type de langage informatique descriptif. Il s'agit plus précisément d'un format de données utilisé dans l'univers d'Internet pour la mise en forme des pages Web. Il permet, entre autres, d'écrire de l'hypertexte, mais aussi d'introduire des ressources multimédias dans un contenu. Développé par le W3C (World Wide Web Consortium) et le WHATWG (Web Hypertext Application Technology Working Group), le format ou langage HTML est apparu dans les années 1990. Il a progressivement subi des modifications et propose depuis 2014 une version HTML5 plus aboutie.

Avantage de progammation HTML

  • Gratuit.
  • Facile à apprendre.
  • La base de tout langage Web.
  • simplification de modification du code.
  • Pris en charge par tous les navigateurs.
  • Peut être facilement intégré à plusieurs langage Web.

Editeurs de code HTML

Un éditeur HTML est un logiciel soit accessible directement en ligne, soit à télécharger, qui aide le développeur à coder les différentes pages d'un site internet. Généralement facile à utiliser, l'éditeur HTML ou éditeur web est une sorte de boîte à outils, qui permet de visualiser rapidement le rendu graphique d'un document.

Balises HTML

Un élément HTML est défini par une balise de début, du contenu et une balise de fin.

<Balise> Le contenu va ici...</Balise>

Les éléments HTML sans contenu sont appelés éléments vides.

La balise <br> définit un saut de ligne et elle est un élément vide sans balise fermante

Les balises basiques

<!Doctype HTML> Definit le type de document en html.

<head> Contient des métadonnées/informations pour le document

<meta> Définit les métadonnées d'un document HTML

<title> Définit un titre pour le document

<body> Définit le corps du document

<h1> vers <h6> Définit les titres HTML.

<p> Définit un paragraphe.

<footer> Définit un pied de page pour un document ou une section.

<!-- --> Définit un commentaire

Les balises d'image

<img> Définit une image

<svg> Définit un conteneur pour les graphiques SVG.

<map> Définit une image cliquable côté client.

<area> Définit une zone à l'intérieur d'une image cliquable.

<figure> Spécifie le contenu autonome.

<picture> Définit un conteneur pour plusieurs ressources d'image

Les balises de lien

<a> Définit un lien hypertexte.

<link> Définit la relation entre un document et une ressource externe.

<nav> Définit les liens de navigation.

Les balises de tableau

<table> Définit un tableau.

<th> Définit une cellule d'en-tête dans un tableau.

<tr> Définit une ligne dans un tableau.

<td> Définit une cellule dans un tableau.

<thead> Regroupe le contenu de l'en-tête dans un tableau

Les balises de listes et formulaires de saisie

<ul> Définit une liste non ordonnée.

<ol> Définit une liste ordonnée.

<li> Définit un élément de liste.

<dl> Définit une liste de description.

<dt> Définit un terme/nom dans une liste de description.

<dd> Définit une description d'un terme/nom dans une liste de description.

<form> Définit un formulaire HTML pour la saisie de l'utilisateur.

<input> Définit un contrôle d'entrée.

<label> Définit une étiquette pour un élément <input>.

<textarea> Définit un champ de saisie multiligne (zone de texte).

<datatlist> Spécifie une liste d'options prédéfinies pour les contrôles d'entrée.

<button> Définit un bouton cliquable.

Attributs HTML

Les attributs HTML sont utilisés dans la balise d'ouverture. Ils fournissent des informations supplémentaires sur les éléments HTML. Un attribut modifie la fonctionnalité par défaut d'un élément ou offre des fonctionnalités à des types d'éléments spécifiques qui ne peuvent pas fonctionner correctement sans eux. L'attribut a un nom, suivi du signe égal (=) et d'une valeur placée entre guillemets (" ").

<Balise attribut="Valeur"> Le contenu va ici...</Balise>

Balise Attribut Exemple
<meta> charset, name, content <meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<img> src, width, height, alt <img width="50px" height="80px" src="/Image/HTML.png" alt="image HTML">
<a> href, target, title <a href"www.google.com" title="Google" target="_blank"><a>
<input> type, name, value, placeholder <input type="text" name="Nom" placeholder="Nom d'utilisateur">
<p> Style <p style="color:red; font-size:10px;"><p>
