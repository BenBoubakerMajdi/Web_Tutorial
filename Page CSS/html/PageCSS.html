C'est quoi CSS en informatique?

Le CSS pour Cascading Style Sheets, est un langage informatique utilisé sur Internet pour la mise en forme de fichiers et de pages HTML. On le traduit en français par feuilles de style en cascade. Apparu dans les années 1990, le CSS se présente comme une alternative à la mise en forme via des balises, notamment HTML. Un peu plus complexe à maîtriser, il permet un gain de temps considérable dans la mise en forme d'une page web par rapport à ces balises. Grâce au CSS, vous pouvez en effet appliquer des règles de mise en forme (titrage, alignement, polices, couleurs, bordures, etc.) à plusieurs documents simultanément.

Avantage de progammation CSS

  • Gratuit.
  • Compatible avec tous les navigateurs.
  • Séparation de contenue et de la mise en forme.
  • Plus grand contrôle sur la présentation de vos pages Web.
  • Permet d'appliquer des styles cohérent aux éléments de votre site,
  • Plus facile d’entretenir votre site internet et de le mettre à jour.

Lien entre CSS et HTML

Avant d'utiliser les nombreuses possibilités offerte par le CSS, il est essentiel de savoir comment l'utiliser dans le cadre d'un projet Web. Nous verrons qu'il existe plusieurs méthodes pour appliquer du style au éléments d'une page et vous apprendrez à vous servir des sélecteurs CSS afin de styliser un ou plusieurs éléments à la fois et selon une hiérarchie et des règles de priorité établies.

Déclaration CSS en ligne

Une première méthode consiste à intégrer le code CSS directement dans votre balise HTML en utilisant l'attribut style. Cette technique est à éviter puisqu'elle ne permet pas de dissocier le contenu de la mise en forme. Vous remarquerez les « : » entre la propriété et la valeur et le « ; » à la fin.
Exemple : <p Style="font-size:2rem ;color:red;">Contenu ici...</p>

Feuille de style interne

Il est possible d'écrire du code CSS dans l'entete d'un document HTML, dans la balise <head>. Il suffit pour cela de l'encadrer par une balise <style> Cette méthode permet de retirer la mise en forme du corps du document, mais elle ne s'appliquera qu'a la page concernée et non à l'ensemble du site.
Exemple :
<Style>
Votre Style ici...
></Style>

Feuille de style externe

L'élément HTML <link> définit la relation entre le document courant et une ressource externe. Cet élément peut être utilisé pour définir un lien vers une feuille de style, vers les icônes utilisées en barre de titre ou comme icône d'application sur les appareils mobiles.
Exemple : <link rel="stylesheet" href="Style.css">

Attributs, fonctions et pseudo-selecteurs CSS

Les CSS Pseudo-Classes

Une pseudo-classe est utilisée pour définir un état spécial d'un élément.

:active Sélectionne l'élément actif

:visited Sélectionne l'élément visité

:hover Sélectionne l'élément au passage de la souris

:focus Sélectionne l'élément qui a le focus

:not(sélecteur) Sélectionne chaque élément qui n'est pas le sélecteur

:nth-of-type(n) Sélectionne chaque élément qui est le niéme élément de son parent

:nth-child(n) Sélectionne chaque élément qui est le niéme enfant de son parent

Les Unités CSS

CSS a plusieurs unités différentes pour exprimer une longueur.

les Longueurs absolues :

Les unités de longueur absolue sont fixes.

Exemple : px(Pixels), in(Inches), cm (Centimeters).

les Longueurs relatives :

Exemple : rem , em , vw(Viewport Width) , vh(Viewport Height)

Les fonctions CSS

Les fonctions CSS sont utilisées comme valeur pour diverses propriétés CSS.

linear-gradient() Crée un dégradé linéaire.

radial-gradient() Crée un dégradé radial.

rgba() Rouge-Vert-Bleu-Alpha (RGBA).

hsla() Hue-Saturation-Lightness-Alpha (HSLA).

Les CSS pseudo-éléments

Un pseudo-élément CSS est utilisé pour styliser des parties spécifiées d'un élément.

::before Insérez quelque chose avant le contenu de chaque élément attribué a cette pseudo-élément.

::after Insérez quelque chose apres le contenu de chaque élément attribué a cette pseudo-élément.

::marker Sélectionne les marqueurs des éléments de la liste

::first-line Sélectionne la première ligne de chaque élément attribué a cette pseudo-élément.

::first-letter Sélectionne la première lettre de chaque élément attribué a cette pseudo-élément.

Des attributs CSS Basique

font-family : Spécifie la famille de polices pour le texte.

font-size : Spécifie la taille de la police du texte.

font-weight : normal, bold, bolder, lighter,

color : Définit la couleur d'un texte.

background-color : Définit la couleur de l'arriére plan.

background-image : Définit une liste de description.

text-align : Définit l'alignement de texte.

text-decoration : Ajoute une décoration au texte

width Définit la largeur d'un élément.

height : Définit la hauteur d'un élément.

border : Définit un bordure.

border-color : Définit la couleur de bordure.

border-style : Définit le style de bordure.

Selecteurs et combinateurs CSS

Un sélecteur CSS est un mot-clef qui permet de désigner une catégorie d'éléments de la page éventuellement de nature différente ou une relation entre deux éléments. On pourra par exemple sélectionner tous les titres de niveau 2 dans le menu, ou encore tous les éléments que l'on a marqués comme étant en rouge. Pour ce faire, les sélecteurs CSS trient les éléments de la page selon leur type, certains de leurs attributs, selon leur imbrication dans le code de la page ou encore selon l'interaction avec l'utilisateur pour les liens par exemple.

Selecteur Exemple Définition
#id #mon_projet Sélectionne l'élément avec id="mon_projet".
.class .mon_projet Sélectionne l'élément avec class="mon_projet".
.class1.class2 .a.b Sélectionne tous les éléments avec à la fois class="a" et class="b".
.class1 .class2 .a .b Sélectionne tous les éléments avec class="b" qui est un descendant d'un élément de class="a".
élément p Sélectionne tous les éléments <p>.
élément,élément div,p Sélectionne tous les éléments <div> et tous les éléments <p>.
élément+élément div+p Sélectionne le premier élément <p> qui est placé immédiatement après les éléments <div>.
élément>élément div>p Sélectionne tous les éléments <p> où le parent est un élément <div>.
élément~élément ul~p Sélectionne chaque élément <p> précédé d'un élément <ul>.
