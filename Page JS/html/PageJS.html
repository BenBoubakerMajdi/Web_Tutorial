C'est quoi JavaScript en informatique?

JavaScript désigne un langage de développement informatique, et plus précisément un langage de script orienté objet. On le retrouve principalement dans les pages Internet. Il permet, entre autres, d'introduire sur une page web ou HTML des petites animations ou des effets. Le langage JavaScript se distingue des langages serveurs par le fait que l'exécution des tâches est opérée par le navigateur lui-même, sur l'ordinateur de l'utilisateur, et non sur le serveur web. Il s'active donc généralement sur le poste client plutôt que côté serveur.

Avantage de progammation JavaScript

  • Facile à comprendre et à apprendre.
  • prise en compte de l’expérience utilisateur.
  • Fournit diverses interfaces aux développeurs.
  • Compatible avec tous les supports numériques.
  • Peut être intégré à plusieurs langage de programmation.
  • Améliore les performances des sites et des applications web.

Lien entre JavaScript et HTML

Javascript s'intègre dans le code HTML (tout comme CSS).Mais pour que le navigateur sache où commence le script et où finit il, il faut le mettre à l'intérieur des délimiteurs du script, la balise <script>, elle permet d'intégrer du Javascript au sein d'un document HTML. Cependant, il n'y a pas que Javascript en tant que langage de script coté client. Il serait donc plus approprié de spécifier le type du langage utilisé.

Exemple :
<script language="javascript"></script>
<script type="text/javascript"></script>

Il y'a deux méthodes pour lier un fichier JavaScript à un fichier HTML.

Liaison externe

On utilise l’attribut "src" dans la balise <script> pour lier le fichier JavaScript au fichier HTML. L’attribut "src" permet de spécifier l’URL ou l’emplacement du fichier JS externe.
Exemple : <script src="PageJS/Main.js"></script>

Liaison interne

Contrairement à la liaison externe de fichier JS, On Ajoute le code de JavaScript dans le fichier HTML à l’aide de la balise <script>. Elle est utilisée pour ajouter un script côté client. On peut écrire le JavaScript en interne à l’intérieur de la balise <script> dans un fichier HTML. Lorsqu'on' exécute le fichier HTML, le JavaScript sera chargé.
Exemple :
<script>
Votre Script ici...
</script>

Où on place les scripts?

Il n'y a pas de restriction sur l'endroit où on peut placer les scripts. On peut les placer n’importe où (avant <html>, dans <head>, dans <body> ou après </html>). Cependant, les développeurs ont pour coutume de placer les scripts Javascript dans la balise <head>.

Variables, évenements et fonctions JavaScript

Evenements JavaScript

Les événements HTML sont des "choses" qui arrivent aux éléments HTML. Lorsque JavaScript est utilisé dans des pages HTML, JavaScript peut "réagir" à ces événements.

Un événement HTML peut être quelque chose faite par le navigateur ou quelque chose faite par un utilisateur. Voici quelques exemples d'événements HTML :

  • Une page Web HTML a fini de se charger
  • Un champ de saisie HTML a été modifié
  • Un bouton HTML a été cliqué
JavaScript vous permet d'exécuter du code lorsque des événements sont détectés. HTML permet d'ajouter des attributs de gestionnaire d'événements, avec du code JavaScript, aux éléments HTML.
Exemple : <element event="Script ici..."></element>

Affichage JavaScript

JavaScript peut afficher les données de différentes manières :

innerHTML : Permet d'écrire dans un élément HTML.

document.write() : Permet d'écrire dans la sortie HTML.

window.alert() : Permet d'écrire dans une boîte d'alerte.

console.log() : Permet d'écrire dans la console du navigateur.

Variable JavaScript

Les variables sont des conteneurs pour stocker des données (stocker des valeurs de données). Il y'a 4 façons de déclarer une variable JavaScript :

  • utilisant var
  • utilisant let
  • utilisant const
  • Ne rien utiliser

Opérateurs JavaScript

Un opérateur est un symbole mathématique qui produit un résultat basé sur deux valeurs (ou variables).

Affectation ('=') : Ceci affecte une valeur à une variable.
Exemple : MaChaine = "Merci"

Addition/Soustraction ('+' '-')
Exemple : 9+3; 20-5;

Multiplication/Division ('*' '/')
Exemple : 9/3; 20*5;

Egalité ('==='): Ceci effectue un test pour voir si deux valeurs sont égales. Il renvoie un résultat True/false (booléen).
Exemple : MonVariable === 4;

Types de données JavaScript

Pour pouvoir opérer sur des variables, il est important de connaître leur type.

JavaScript a des types dynamiques. Cela signifie que la même variable peut être utilisée pour contenir différents types de données :

Chaine : Une chaîne (ou une chaîne de texte) est une série de caractères. Les chaînes sont écrites avec des guillemets. On peut utiliser des guillemets simples ou doubles

Nombre : JavaScript n'a qu'un seul type de nombres. Les nombres peuvent être écrits avec "," ou sans décimales

Booléens : Les booléens ne peuvent avoir que deux valeurs : true ou false.

Tableaux : Les tableaux JavaScript sont écrits entre crochets. Les éléments du tableau sont séparés par des virgules.

Objets : Les objets JavaScript sont écrits avec des accolades {}. Les propriétés d'objet sont écrites sous la forme de paires nom:valeur, séparées par des virgules.
